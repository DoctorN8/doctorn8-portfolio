Using Machine Learning to Predict Music Ensemble Placement: Harnessing AI for Harmonic Success
I developed a machine learning (ML) system to predict acceptance into the VMEA Honor Choir with 85% accuracy. The project involved comprehensive data analysis of audition scores, statistical testing using ANOVA and Tukey's HSD, and the implementation of multiple ML models. The final Random Forest Classifier achieved outstanding performance metrics (82% precision, 78% recall) while providing valuable insights into key acceptance factors.

Skills Demonstrated:
  • Machine Learning (Random Forest Classification)
  • Statistical Analysis (ANOVA, Tukey's HSD)
  • Python (Scikit-learn, Pandas)
  • Data Visualization (Matplotlib, Seaborn)
  • Feature Engineering
  • Predictive Modeling
This project isn't just about data; it's about illuminating the path to success in the world of music.”

Boosting Workplace Excellence: The Power of Achievement Goal Orientation in Grading Attitudes
Understanding the interplay between achievement goal orientation (AGO) and attitudes toward grading practices is essential for creating a positive educational environment that promotes success. In this study, we conducted cluster analysis to identify five homogeneous user personas among pre-service music teachers.
  • Non-Mastery-Oriented
  • Non-Performance-Oriented
  • Performance-Oriented
  • Eclectic
  • Mastery-Oriented 
Our findings suggest that a combination of high mastery-based and low performance-based orientations is the most effective in promoting positive attitudes toward grading practices. By applying these insights to training and development programs, businesses can foster a culture of excellence that encourages growth and achievement.
This information can be useful in developing interventions to improve attitudes toward grading practices.”

Discovering Secrets from YouTube: Using Machine Learning to Uncover Trends and Help Businesses Grow
I conducted an intriguing research project using statistical analysis to explore the relationship between YouTube subscribers and revenue in museum channels. Using data from CauseIQ, I performed logarithmic transformations to build a linear regression model. The analysis revealed that doubling YouTube subscribers correlates with a 2.347x increase in revenue, explaining approximately 35% of revenue variance.
